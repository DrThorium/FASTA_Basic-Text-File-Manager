Detailed Description

Implementation of the Huffman Tree

Huffman is an algorithm that takes a frequency table from a text and returns a codification table using a Binary Tree Ordered (Using Heapify) and a simple translation (Tree/Heap -> Codification table) method. Example: Text: Mr. Dr. Thorium. Frequency Table: D:1 R:3 T:1 H:1 O:1 I:1 U:1 M:2 (Which is a std::map (char, int)).

  • Huffman Tree (Heap Representation): TOP->(D, T, H, O, I, U, M, R) Codification Table (Returned): R: 10, M: 01, H: 00, O: 110, I: 1111, U: 11101, D: 111001, T: 111000. The codification value is a simple vector of ints (1,0,) that represents a bits stream. So, from the original text: Mr. Dr. Thorium (we're excluding spaces and "." for the example), MRDRTHORIUM we will have using the Huffman encoding the binary set: 011011100110111000001101011111110101+0000 (we complete the Final Zeros) Which is only 36(+4=40) bits long (5 bytes). If we use the char original bit stream we will have: 0100110101010010010001000101001001010100010010000100111101010010010010010101010101001101 Which is 88 bits long (11 bytes). Therefore, for the last example we have a 1:2 Compression Rate.
